## 1️⃣ What is the difference between var, let, and const?
Ans: var is function-scoped and can be re-declared and reassigned.
<br> let is block-scoped and can be reassigned but not re-declared. 
<br> const is block-scoped and cannot be reassigned or re-declared.

## 2️⃣ What is the spread operator (...)?
Ans: The spread operator (...) expands an array or object into individual elements.

## 3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map() — transforms each element and returns a new array. <br>
filter() — returns a new array with elements that pass a condition. <br>
forEach() — just loops through elements and act according to what code comands and returns nothing.

## 4️⃣ What is an arrow function?
Ans: A shorter syntax for writing functions. Ex:- <br>
// Regular<br>
function add(a, b) { return a + b; }<br>
// Arrow <br>
const add = (a, b) => a + b;

## 5️⃣ What are template literals?
Ans: Strings that are inside backticks (` `) instead (" "). We can put variables directly inside the string using ${}.
